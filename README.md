# c-internals-lab

A collection of low-level C experiments focused on understanding how software interacts with the system. This repository is intended for learning, research, and practical exploration through small, focused programs.

## Overview

This project serves as a personal workspace for exploring core concepts in low-level programming using C. The code emphasizes simplicity, direct interaction with the system, and minimal abstractions.

The objective is to build a deeper understanding of program behavior by working close to the execution environment and observing how different components operate in practice.

## Requirements

* A C compiler (MSVC or MinGW)
* A Windows x64 environment (recommended)
* Basic familiarity with C programming

## Usage

Each example can be compiled independently following the build.txt in each folder.

This repository is provided for educational and research purposes only. The content is meant to support learning and experimentation in controlled environments.
